# Panduan Setup Arlogic

1. Buka terminal di folder `arlogic`.
2. Jalankan `npm install`.
3. Salin file `.env.example` menjadi `.env`.
4. Isi variabel:
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
5. Jalankan `npm run dev`.
6. Akses di `http://localhost:3000`.

> Jika Anda tidak menggunakan Supabase, aplikasi ini dapat dijalankan sebagai mock UI tanpa login backend.
